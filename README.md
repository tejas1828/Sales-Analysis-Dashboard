# Super Store Sales Dashboard (Power BI)
# Project Objective
This project presents an interactive Sales Dashboard built using Power BI to analyze and visualize retail store performance across multiple dimensions such as category, region, time, and payment modes.
Dataset

## The dataset used for this project is the Superstore Sales Dataset, commonly used for data analysis and visualization practice.

### Download Dataset:
-<a href="https://github.com/tejas1828/Sales-Analysis-Dashboard/blob/main/SuperStore_Sales_Dataset.csv">Dataset</a>
## Key Features
•	Overall Sales Overview 
o	Displays total sales (341K) for quick performance insight. 
•	Sales by Category 
## Compares performance across: 
	Office Supplies 
	Furniture 
	Technology 
•	Sales by Sub-Category 
o	Detailed breakdown of top-performing items like Chairs, Binders, and Phones. 
•	Regional Analysis 
o	Interactive filter for regions: Central, East, South, West 
o	Map visualization showing Sales and Profit by State 
•	Time-Based Analysis 
o	Sales by Year-over-Year (YoY) trend 
o	Profit by YoY comparison 
•	Customer Segmentation 
o	Sales distribution across: 
	Consumer (48%) 
	Corporate (33%) 
	Home Office (20%) 
•	Payment Mode Insights 
o	COD: 41% 
o	Online: 39% 
o	Cards: 20% 
•	Sales by Region 
o	Donut chart highlighting contribution (currently Central 100%) 
 ## Tools & Technologies Used
•	Power BI (Data Visualization) 
•	DAX (Data Analysis Expressions) 
•	Data Modeling & Transformation 
🎯 Purpose
## The dashboard helps stakeholders:
•	Identify top-performing categories and products 
•	Analyze regional sales trends 
•	Understand customer segments and payment preferences 
•	Make data-driven business decisions
